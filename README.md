# MediaLab Documents

**MediaLab Documents** is a **JavaFX-based** document management that was developed as part of the **Multimedia Technologies (MediaLab)** coursework assignment.

---

## How to Run

### 1) Clone

git clone https://github.com/ntua-el21830/File-browser

cd media_lab

### 2) Build
mvn clean compile

### 3) Run
mvn javafx:run

---
## Technologies Used

Java (JDK 21)

JavaFX

Maven

Jackson (JSON persistence)
